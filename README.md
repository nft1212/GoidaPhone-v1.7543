(EN)GoidaPhone is a local network voice and text communication application with Windows XP style interface. The program allows users within the same local network to communicate through voice calls and text messages without requiring internet connection. It features automatic peer discovery using UDP broadcast messages and establishes direct TCP connections for voice communication. The application provides authentic Windows XP styling with classic blue title bars, beveled buttons, and system colors. Users can select specific audio input and output devices, test microphone and speakers, adjust volume levels, and see real-time user presence in the network. The system uses PyAudio for audio processing with configurable sample rates and chunk sizes. Network communication includes periodic presence broadcasting, user status monitoring, and handling of both incoming and outgoing voice calls with proper connection management and error handling. The interface includes a user list panel, chat message area, call control buttons, and status indicators with support for Windows notifications when receiving messages while the application is not focused.


(RU)GoidaPhone это приложение для голосовой и текстовой связи по локальной сети с интерфейсом в стиле Windows XP. Программа позволяет пользователям в одной локальной сети общаться через голосовые звонки и текстовые сообщения без необходимости подключения к интернету. Приложение автоматически находит других пользователей с помощью UDP-широковещательных сообщений и устанавливает прямые TCP-соединения для голосовой связи. Интерфейс выполнен в аутентичном стиле Windows XP с классическими синими заголовками окон, рельефными кнопками и системными цветами. Пользователи могут выбирать конкретные аудиоустройства для ввода и вывода, тестировать микрофон и динамики, регулировать уровень громкости и видеть присутствие пользователей в сети в реальном времени. Система использует PyAudio для обработки звука с настраиваемой частотой дискретизации и размером блоков. Сетевое взаимодействие включает периодическую широковещательную рассылку присутствия, мониторинг статуса пользователей и обработку как входящих, так и исходящих голосовых вызовов с правильным управлением соединениями и обработкой ошибок. Интерфейс включает панель списка пользователей, область текстового чата, кнопки управления звонками и индикаторы статуса с поддержкой уведомлений Windows при получении сообщений, когда приложение не активно.
